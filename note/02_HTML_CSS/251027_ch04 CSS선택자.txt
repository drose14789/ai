4장 CSS 선택자
1. 선택자: HTML 문서 중 특정 부분을 선택하여, 해당 부분의 속성을 변경하는 목적으로 사용 
   (CSS를 사용하는 이유, Javascript, jQuery등에서도 사용)

2. tag 선택자: 특정 태그를 선택하여 CSS속성을 적용할 수 있다

3. CSS 선택자: 웹페이지의 특정부분을 선택하여, 태그의 속성을 변경하는 목적 이용 

4. 선택자: 전체라는 의미의 * 선택자를 사용하면 문서 전체를 선택하는 의미가 있다

5. id(#)와 class(.) : 태그의 id와 clas를 이용하여 CSS에 속성을 설정할 수 있음
