4장 CSS 선택자
6. tag, id, class 혼합 : tag, id, class를 혼합하여 CSS에서 속성을 설정할 수 있음

7. 속성 선택자 : tag의 속성을 선택하여 CSS의 속성을 설정할 수 있음

8. 후손 및 자손 선택자 : tag의 속성을 선택하여 CSS의 속성을 설정할 수 있음

9. 동위(=형제) 선택자(+하나만, ~다선택) : 태그의 동등한 위치를 판단하여 CSS속성을 설정할 수 있음

10. 반응 선택자 : 마우스를 올려 놓았을 때(hover상태)의 반응에 따른 속성을 설정할 수 있음

11. 상태 선택자 : 상태에 따라 CSS속성이 변화하는 설정을 할 수 있음  
     ex. input태그의 경우 focus가 맞혀졌을 때

12. 구조 선택자 : 구조에 따라 CSS 속성이 변화하는 설정을 할 수 있음

13. 문자 선택자

14. 링크 선택자 : 문서에 링크(href) 되어 있는 문자를 선택하여, CSS 속성을 설정할 수 있음

15. 부정 선택자 : 나를 제외한 모든 태그에 CSS 속성을 적용할 수 있음
