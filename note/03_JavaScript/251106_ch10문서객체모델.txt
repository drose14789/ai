10장 문서 객체 모델
1. 문서 객체 모델과 관련된 용어 정리 ; 태그, 문서객체, 노드
1) 태그를 자바스크립트에서 이용할 수 있는 객체로 만든 것이 문서 객체.
2) 트리 모양의 문서 객체 모델로 나타낼 수 있고 각 요소를 노드라 부른다
3) HTML 페이지에 적혀 있는 태그를 읽으며 생성하는 것을 정적으로 문서 객체를 생성한다고 하고, 
   HTML 원 페이지에는 없던 문서 객체를 생성하는 것을 동적으로 문서 객체를 생성한다고 함

2. 문서 객체 만들기 : 텍스트 노드를 갖는 문서 객체 만들기
1) 동적으로 문서 객체를 만들기 위해
  ① Document 객체의 노드 생성 
      - createElement(tagName) : 요소 노드를 생성
      - createTextNode(text) : 텍스트 노드를 생성
  ② 문서 객체 연결:
      - appendChild(node) : 객체에 노드를 연결
2) 텍스트 노드를 갖지 않는 문서 객체 만들기

4. 문서 객체 가져오기 : 
- 이미 존재하는 HTML 태그를 자바스크립트로 가져오는 방법 중 id로 가져오기
- 태그이름이나 이름으로 가져오기
1) getElementById(id) 태그의 id속성이 id와 일치하는 문서 객체를 가져옴
2) getElementsByClassName(“class명”) : 시간 뿌리기
3) getElementsByName(name): 
 태그의 name 속성이 name과 일치하는 문서 객체를 배열로 가져온다
4) getElementsByTagName(‘태그명’);
5) document.querySelector(선택자) ; 선택자로 가장 처음 선택되는 문서 객체를 가져온다
6) document.querySelectorAll(선택자); 선택자로 선택되는 문서 객체를 배열로 가져옵니다
