5장 함수
1. 익명 함수
- 함수는 코드의 집합을 나타내는 자료형. 익명 함수는 다음 형태로 만듬

// 화살표 함수 및 class는 ECMA scipt 6
- var functionVar = function() { }; // 함수 표현식
- var functionVar = () => {}; // 화살표 함수

// 매개변수 지정 방법
- () => { . . . } // 매개변수가 없을 경우
- x =>{ . . . } //매개변수가 한 개인 경우, 소괄호를 생략할 수 있음
- (x, y) => { . . . } // 매개변수가 2개이상인 경우, 소괄호를 생략할 수 없음

// 함수 몸체 지정 방법
- x => { return x*x } // single line block
- x => x * x         // 함수 몸체가 한줄의 return 구문이라면 중괄호 생략 가능 
- () => { return { a: 1 }; }
- () => ({ a: 1 })  // 위 표현과 동일. 객체 반환시 소괄호를 사용
- () => {           // multi line block.
  const x = 10;
  return x * x;
 };

// https://poiemaweb.com/es6-arrow-function에서 참조바람 
※ 익명 함수 생성과 출력 ; 이름이 없으므로 변수에 넣어 사용해야 함
※  내장함수는 코드를 볼 수 없게 막아 놓음. 
    이름이 있는 함수를 ‘선언적 함수’라 부름. 
    함수의 이름이 없는 함수를 ‘익명함수’라 부름.
※ 자바스크립트에서 함수는 하나의 자료형이지만 
   여타 자료형과 다르게 뒤에 괄호를 열고 닫음으로써 코드를 실행함 
   이를 함수를 호출한다고 표현함.

4. 매개변수
- 자바 스크립트는 함수를 생성할 때 지정한 매개변수보다 많거나 적은 매개변수를 
   사용하는 것을 허용함. 원래 함수에서 선언된 매개변수보다 많게 사용하면 일반적으로 
   추가된 매개변수는 무시. 원래 함수에서 선언한 매개변수보다 적게 사용하면 
   지정하지 않는 매개변수는 undefined로 입력됨
※ Array()함수 ; 빈 배열을 만듬
   Array(number) ; 매개변수만큼의 크기를 가지는 배열을 만듬
   Array(any, …, any); 매개변수를 배열로 만듬

5. 가변 인자 함수 ; 매개변수를 선언된 형태와 다르게 사용했을 때,
