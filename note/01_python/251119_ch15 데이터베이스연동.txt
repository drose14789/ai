1절. SQLite 데이터베이스 연결
- SQLite 데이터 베이스는 별도의 DBMS 없이 SQL문 이용해서 DB액세스를 할 수 있도록 만든 
  간단한 디스크 기반 DB
- C라이브러리
- SQLite는 프로토타입 생성시 사용
- 프로젝트 : 분석 -> 설계 -> 구현 -> 테스트 -> 고객에게 배포
              프로토타입(SQLite)    완제품(Oracle, MySQL, Maria,...)
- DB browser for SQLite

1.2 데이터베이스 연결
(1) DB 연결 : sqlite로 연결시 DB파일이 없으면 빈 DB 생성, 있으면 연결
(2) SQL 전송 객체(cursor 객체)
(3) SQL 전송(cursor.execute()이용)
- insert, update, delete문 실행 결과: cursor.rowcount
- select문 실행 결과:
            fetchall() : 결과를 모두 받을 때(튜플 list)
            fetchone() : 결과를 한행씩 받을 때(튜플)
            fetchmany(n) : 결과를 n행 받을 때(튜플 list)
(4) cursor 해제 & 데이터베이스연결객체 해제
